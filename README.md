# Global Kids International School 🏫

Welcome to the Global Kids International School official website repository. 

This repository contains the source code for our responsive, modern, and beautiful landing page. It is built using clean HTML5, CSS3, and vanilla JavaScript without relying on heavy frameworks, ensuring blazing-fast load times and seamless cross-device compatibility.

## 🌟 Features
- **Vibrant & Modern Design:** A beautifully styled user interface using a Royal Blue & Coral color palette.
- **Fully Responsive:** Perfectly optimized for all devices, from ultra-wide desktops to mobile phones.
- **Dynamic Interactions:** Smooth scroll, intersection observer animations, floating elements, and counting statistics.
- **Real-World Imagery:** High-quality integrated photography for campus facilities, events, and teachers.
- **SEO & Accessibility Ready:** Proper Open Graph tags, aria labels, and semantic HTML structure.

## 🚀 Deployment
This website is designed to be hosted seamlessly on **GitHub Pages**.

1. Navigate to the **Settings** tab of this repository.
2. Select **Pages** from the left sidebar.
3. Under **Source**, select **GitHub Actions**.
4. GitHub will automatically build and deploy the `index.html` file to your public URL.

## 📂 Structure
- `index.html`: The main structural entry point.
- `styles.css`: All responsive layouts, color tokens, typography, and CSS animations.
- `script.js`: Navbar scroll effects, stat counters, mobile menu, and scroll animations.
